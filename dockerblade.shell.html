dockerblade.shell module¶

class dockerblade.shell.Shell(container, path, sources=NOTHING, environment=NOTHING)[source]¶

Bases: object

Provides shell access to a Docker container. Do not directly call the constructor to create shells. Instead, use the :method:`shell` method of Container to build a shell.

container: Container

The container to which the shell is attached.

path: str

The absolute path to the binary (inside the container) that should be used to provide this shell.

_sources: Sequence[str]

A sequence of files, given by their absolute paths, that should be sourced by the shell upon construction.

_environment: Mapping[str, str]

A mapping from the names of environment variables to their values.

ContainerFileNotFound

If a given source file is not found.

run(args, *, encoding='utf-8', cwd='/', text=True, stdout=True, stderr=False, time_limit=None, kill_after=1)[source]¶

Executes a given command and blocks until its completion.

args: str

The command that should be executed.

encoding: str

The encoding that should be used for decoding, if the output of the process is text rather than binary.

cwd: str

The absolute path of the directory in the container where this command should be executed.

text: bool

If True, the output of the process is decoded to a string using the provided :param:`encoding`. If False, the output of the process will be treated as binary.

stderr: bool

If True, the stderr will be included in the output.

stdout: bool

If True, the stdout will be included in the output.

time_limit: int, optional

The maximum number of seconds that the command is allowed to run before being terminated via SIGTERM. If unspecified, no time limit will be imposed on command execution.

kill_after: int

The maximum number of seconds to wait before sending SIGKILL to the process after attempting termination via SIGTERM. Only applies when time_limit is specified.

CompletedProcess

A summary of the outcome of the command execution.

Return type

CompletedProcess
